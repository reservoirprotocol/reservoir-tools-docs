Reservoir NFT is a **complete NFT marketplace and API that allows developers to interact with the NFT market using a single toolkit**. With the tools, you can build custom NFT marketplaces, add NFT data into your app, or get a full suite of NFT infrastructure on your chain - from seaport deployment to an NFT marketplace frontend.

The product includes:

<AccordionGroup>
  <Accordion title="Seaport Deployment">
    For custom chain support, Reservoir offers deployment services of the
    Seaport Exchange, a best-in-class decentralized NFT exchange
  </Accordion>
  <Accordion title="NFT Trading and Data APIs">
    Reservoir NFT provides complete suite of token data and trading apis,
    including NFT minting & buying, bidding & listing. Explore the API [here]().
  </Accordion>
  <Accordion title="ReservoirKit">
    ReservoirKit is a React library that makes it easy to add marketplace
    functionality into any app. The kit has out of the box and customizable
    modals for major market actions, as well as a series of useful hooks to
    simplify development. Learn more about ReservoirKit
    [here](ref:reservoirkit).
  </Accordion>
  <Accordion title="Reservoir Market (FE)">
    Reservoir Market is a complete NFT marketplace frontend that is made
    available for any chain that Reservoir supports
